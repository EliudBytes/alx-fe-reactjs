# Tailwind React Integration Project

This project is a React app created with Vite and styled using Tailwind CSS.

## Setup

- Created with: `npm create vite@latest tailwind-react-integration -- --template react`
- Tailwind CSS installed and configured using `@tailwindcss/vite` plugin
- Tailwind directives added to `src/index.css`

## Running the app

```bash
npm install
npm run dev
